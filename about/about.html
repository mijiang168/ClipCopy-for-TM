ClipCopy for Tampermonkey 0.0.1

Thank you for installing this extension!

Short description

This program is an add-on for Tampermonkey allowing an userscript to copy text to the clipboard using GM_setClipboard API.

Use instructions

This is still a beta version (not active yet!)
